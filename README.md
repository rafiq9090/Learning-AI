# 🩺 Diabetes Prediction AI System

This project is a Machine Learning based classification system that predicts whether a patient has diabetes based on medical diagnostic measurements.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Joblib
- **Platform:** Google Colab

## 📊 Models Evaluated
| Model | Accuracy |
| :--- | :--- |
| Logistic Regression | 75.32% |
| Random Forest | 72.08% |

## 🧠 Key Learnings
- **Data Preprocessing:** Handled feature scaling using `StandardScaler`.
- **Model Inference:** Implemented a prediction script for new patient data.
- **Model Persistence:** Saved models as `.pkl` files for future API integration (FastAPI/Go).

## 🚀 How to Run
1. Clone the repo.
2. Install dependencies: `pip install pandas scikit-learn joblib`.
3. Run the notebook in Google Colab or VS Code.
